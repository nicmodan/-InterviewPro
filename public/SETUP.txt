STEP 1 — Firebase Console
  Go to: console.firebase.google.com
  Project: modanmic-ai-power-exams

  A) Authentication → Sign-in method → Email/Password → ENABLE

  B) Firestore Database → Create database (production mode)

  C) Firestore → Rules tab → Replace ALL rules with:

─────────────────────────────────────────────
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // Admin accounts
    match /admins/{adminId} {
      allow read, write: if request.auth != null
                         && request.auth.uid == adminId;
    }

    // Unite 13 portal config (fee, Monnify keys)
    // PUBLIC READ — so index.html can fetch fee without login
    // WRITE restricted to authenticated admins only
    match /portal_config/{docId} {
      allow read:  if true;
      allow write: if request.auth != null;
    }

    // Unite 13 Adeota registrations
    match /adeota_registrations/{regId} {
      allow create: if true;
      allow read, update, delete: if request.auth != null;
    }

    // Original InterviewPro collections
    match /applications/{appId} {
      allow create: if true;
      allow read:   if request.auth != null;
      allow update: if true;
    }
    match /payments/{payId} {
      allow create: if true;
      allow read:   if request.auth != null;
    }
  }
}
─────────────────────────────────────────────

  D) Click "Publish" to save the rules.

  E) Firestore → Indexes → Add composite index (for admin dashboard):
     Collection: adeota_registrations
     Fields:     submittedAt (Descending)
     Click "Create" — takes ~2 minutes to build.
     (Without this, the dashboard may show an empty list initially)

===================================================
STEP 2 — Deploy
  Option A (Netlify — Recommended):
    Go to netlify.com/drop
    Drag and drop this entire folder
    Done! Site live in ~30 seconds.

  Option B (Firebase Hosting):
    npm install -g firebase-tools
    firebase login
    firebase init hosting  (public dir = this folder, SPA = No)
    firebase deploy

===================================================
STEP 3 — Create your Admin Account
  1. Open your deployed site
  2. Go to /register.html
  3. Fill in your details → Create Account
  4. You are redirected to the InterviewPro dashboard (login.html)
  5. To access the Unite 13 Adeota admin panel, navigate to:
     https://yoursite.com/adeota-admin.html

===================================================
STEP 4 — Share the Registration Form
  Share the index.html URL with school proprietors:
  https://yoursite.com/index.html

  OR copy it from the Settings tab in adeota-admin.html

  Proprietors do NOT need an account.
  They open the link, fill in the 4-step form, and submit.

===================================================
ADEOTA ADMIN DASHBOARD (adeota-admin.html)
  Overview Tab:
    • Total schools registered
    • Total candidates
    • States represented
    • 7-day registration chart
    • School type breakdown (pie chart)
    • Recent registrations table

  School Registrations Tab:
    • Full table of all registered schools
    • Search by school name or proprietor name
    • Filter by state
    • Click "View →" to see full school + proprietor details
      and the complete candidate list for that school
    • Export to Excel button

  All Candidates Tab:
    • Flat view of every candidate across all schools
    • Filter by name, class, or gender
    • Export to Excel button

  Settings Tab:
    • Edit portal title and instructions
    • Set registration deadline
    • Copy the registration form URL

  Sidebar → "Export All Data":
    • Downloads a single Excel file with two sheets:
      Sheet 1 = Schools, Sheet 2 = Candidates

===================================================
REGISTRATION FORM FEATURES (index.html)
  Step 1 — Proprietor Details:
    Full name, gender, DOB, NIN, phone (×2), email, address, state

  Step 2 — School Information:
    Name, type, ownership, year est., reg. no., phone, email,
    full address, enrollment, notes

  Step 3 — Candidates:
    • Import via Excel (.xlsx / .xls / .csv)
      Excel columns: Surname | First Name | Other Names |
                     Gender | Date of Birth | Class | Subject
    • Download blank template button (auto-generates correct format)
    • Add manually one-by-one with live table preview
    • Remove individual candidates with ✕ button

  Step 4 — Review & Submit:
    • Full summary of all entered data
    • Declaration checkbox
    • Saves to Firestore collection: adeota_registrations
    • Displays unique reference code (e.g. U13-ABC12345)
    • Confetti celebration on success ✨
